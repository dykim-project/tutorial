1. git setting

    2. git init
    레포지토리로 설정한다
    .git 이라는 폴더가 생성된다. 버전 정보가 저장되어있다.

    git add (파일명)
    git 이 해당 파일을 추적하게 설정한다.
    커밋 대기상태가 된다

    git commit
    해당 파일을 버전관리 한다.
    commit 문구를 적는다.

  2. log

    3. git log
    리포지토리의 버전관리 로그를 보여준다.

    git log -p
    수정된 파일내용을 보여준다.

    git log (커밋의 고유 아이디)
    커밋의 고유아이디 이전버전들의 수정된 파일 내용을 보여준다

    git diff (커밋아이디)...(커밋아이디)
    두 커밋을 내용을 비교한다.
